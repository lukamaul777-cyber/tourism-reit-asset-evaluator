# Screenshots Guide

This folder is reserved for manually captured screenshots used in the project README, portfolio materials, and presentation documents.

## Recommended Screenshots

- `home.png`: project overview, KPI cards, score ranking
- `asset_profile.png`: asset profile and gatekeeper result
- `indicator_framework.png`: reference-based indicator framework
- `reit_fit_score.png`: score breakdown and module radar chart
- `risk_warning.png`: risk radar and warning cards
- `model_validity.png`: reliability/validity/robustness checks
- `scenario_simulator.png`: stress-test controls and KPI outputs
- `report_generator.png`: generated report preview and download buttons

## Screenshot Tips

- Use a clean browser zoom level, around 90%-100%.
- Select one representative asset.
- For the scenario simulator, use the Stress Case preset.
- For the report generator, include scenario simulation in the report.
- Avoid exposing local file paths or personal information.
- Save images as PNG.
