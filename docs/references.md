# References

The project reference library is maintained in machine-readable form in `config/model_references.yml`.

- CSRC public infrastructure REITs guidelines.

These references support the gatekeeper layer, especially ownership or operation rights, operating history, stable cash flow, market-based cash-flow source, subsidy dependence, legal compliance, and continuous-operation risk.
